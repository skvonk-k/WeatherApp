# WeatherApp

Консольное приложение на С# (.NET) для получения текущей погоды по названию города.

## Возможности

- Ввод названия города на русском или английском
- Определение координат города через OpenWeatherMap Geocoding API
- Получение текущей погоды (температура, описание погодных условий) через OpenWeatherMap Current Weather
- Вывод температуры в градусах Цельсия и описания погодных условий на русском языке

## Технологии

- С# | .NET 8
- HttpClient - запросы к внешним API
- System.Text.Json - чтение JSON-ответов
- Microsoft.Extensions.Configuration.UserSecrets - безопасное хранение API-ключа

## Настройки перед запуском

1. Зарегистрируйтесь на сервисе OpenWeatherMap и создайте API-ключ;
2. В папке проекта выполните команды:

``` bash
	dotnet user-secrets init
	dothet user-secrets set "WeatherApi:ApiKey" "ваш_API_ключ"
```

3. Запустите проект.

## Пример использования

Введите название города: _Москва_

Погода в городе Москва:
